# トレード記録アプリ セットアップガイド

---

## ステップ1：データ移行（migrate_entries.py）

### 1-1. ライブラリをインストール

コマンドプロンプトで実行：

```
py -m pip install gspread google-auth google-auth-oauthlib
```

### 1-2. まず dry-run で確認

```
cd C:\Users\81805\Documents\Claude\trade_app
py migrate_entries.py --dry-run
```

ブラウザが開いてGoogleアカウントへのログインが求められます。
スプレッドシートのオーナーアカウントでログインしてください。

**確認ポイント：**
- `トレード件数: 67件`（または実際の件数）と表示されること
- `フィールド対応チェック` で ✓ が多いこと
- 最初の3件のプレビューが正しく表示されること

### 1-3. 本番実行（バックアップ自動作成）

問題なければ本番実行：

```
py migrate_entries.py
```

実行後、Google Sheetsで確認：
- Entriesシートが行形式になっているか（1行目=ヘッダー、2行目以降=トレード）
- `Entries_backup_YYYYMMDD_HHMMSS` シートが自動作成されているか

## ステップ2：Google Apps Script にコードを貼り付け

### 2-1. Apps Script を開く

1. ブラウザで [Google Sheets](https://docs.google.com/spreadsheets/d/1wPbALJwAZs7gUGs7anNvTqql2udX8lsMP51zSsHSkAM) を開く
2. メニューバー → **「拡張機能」** → **「Apps Script」**

### 2-2. Code.gs を貼り付け

1. 左のファイル一覧に `コード.gs`（または`Code.gs`）が表示されている
2. クリックして開く
3. **全選択（Ctrl+A）→ 削除**
4. `C:\Users\81805\Documents\Claude\trade_app\Code.gs` の内容を全コピーして貼り付け
5. **Ctrl+S** で保存

### 2-3. Index.html を追加

1. 左のファイル一覧の **「＋」** ボタンをクリック
2. **「HTML」** を選択
3. ファイル名に **`Index`** と入力（拡張子の`.html`は自動付与）
4. 作成されたファイルの内容を全削除
5. `C:\Users\81805\Documents\Claude\trade_app\Index.html` の内容を全コピーして貼り付け
6. **Ctrl+S** で保存

## ステップ3：WebApp としてデプロイ

### 3-1. デプロイ設定

1. 右上の青い **「デプロイ」** ボタン → **「新しいデプロイ」**
2. 歯車アイコン → **「ウェブアプリ」** を選択
3. 以下の設定にする：
   - **説明**: `トレード記録 v1`
   - **次のユーザーとして実行**: `自分（自分のメールアドレス）`
   - **アクセスできるユーザー**: `自分のみ`（または信頼できるGoogleアカウント）
4. **「デプロイ」** ボタンをクリック

### 3-2. アクセス許可

初回のみ、権限確認ダイアログが表示されます：

1. 「権限を確認」をクリック
2. Googleアカウントを選択
3. 「詳細」→「（プロジェクト名）（安全ではないページ）に移動」をクリック
4. 「許可」をクリック

⚠️ 「安全ではない」と表示されますが、**自分で作成したコード**なので問題ありません。

### 3-3. URL をコピー

デプロイ完了後に表示される **ウェブアプリのURL** をコピーしてください。
（例: `https://script.google.com/macros/s/AKfycby.../exec`）

## ステップ4：スマホのホーム画面に追加

### iPhone の場合

1. **Safari** でウェブアプリURLを開く（Chromeでは追加できないので注意）
2. 下の共有ボタン（□に↑）をタップ
3. **「ホーム画面に追加」** をタップ
4. 名前を「トレード記録」にして **「追加」**

### Android の場合

1. **Chrome** でURLを開く
2. 右上メニュー（⋮）→ **「ホーム画面に追加」**
3. 「追加」をタップ

ホーム画面にアイコンが追加され、フルスクリーンで動作します。

## トラブルシューティング

### 「スクリプトの実行権限がない」エラー

→ デプロイ設定の「次のユーザーとして実行」が **「自分」** になっているか確認

### データが表示されない

→ `Code.gs` の `SPREADSHEET_ID` が正しいか確認：
```
1wPbALJwAZs7gUGs7anNvTqql2udX8lsMP51zSsHSkAM
```

### 移行後にデータが0件

→ Google SheetsでEntriesシートを直接確認。ヘッダー行（1行目）と
  データ行（2行目以降）があるか確認。

### スマホで画像アップロードできない

→ iOSの場合はSafariで開いてください。Chromeではカメラ権限に制限があります。

## コードの更新方法

機能を追加したい場合：
1. `Code.gs` や `Index.html` を修正
2. Apps Script で「デプロイ」→ **「デプロイを管理」**
3. 鉛筆アイコン→ バージョンを **「新しいバージョン」** に変更
4. **「デプロイ」** → URLは変わらずそのまま使える
